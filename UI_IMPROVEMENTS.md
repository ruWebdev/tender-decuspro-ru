## Полный путь поставщика

1. **Регистрация** → Выбор роли "Поставщик"
2. **Главная страница** → Просмотр активных закупок
3. **Карточка тендера** → Кнопка "Откликнуться"
4. **Страница откликнуться** → Заполнение цен и комментариев
5. **Отправка предложения** → Сохранение предложения
6. **Просмотр статуса** → "Мои предложения" в навигации

## Полный путь заказчика

1. **Регистрация** → Выбор роли "Заказчик"
2. **Личный кабинет** → "Мои закупки"
3. **Создание закупки** → Заполнение данных и позиций
4. **Список закупок** → Просмотр созданных закупок
5. **Просмотр тендера** → "Просмотреть предложения"
6. **Список предложений** → "Выбрать победителем"
7. **Страница выбора** → Выбор лучшего предложения
8. **Сравнение** → "Сравнить предложения"
